# Access Request Procedure
## TSHLA Medical

**Last Updated:** January 9, 2026
**HIPAA:** ยง164.308(a)(4) - Access Authorization

## Overview
Procedure for requesting additional system access or elevated privileges.

## Request Process

### 1. Submit Request
- Email: privacy-officer@tshla.ai
- Include: Name, current role, requested access, business justification, duration needed

### 2. Review (1-2 business days)
- Privacy Officer reviews against Minimum Necessary rule
- Technical feasibility assessed
- Risk evaluation performed

### 3. Approval/Denial
- Super Admin makes final decision
- Written notification sent to requester
- Documented in access control log

### 4. Implementation (if approved)
- Access granted with specified expiration date
- User notified of new permissions
- Training provided if needed

### 5. Review
- Temporary access reviewed at expiration
- Permanent access reviewed quarterly

## Emergency Access
- Call: [Emergency Line]
- Break-glass access available
- Must file incident report within 24 hours

## Appeal Process
If denied, requester may appeal to Privacy Officer within 5 business days with additional justification.
